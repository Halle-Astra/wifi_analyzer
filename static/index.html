蓝牙设备（2.4GHz 直接干扰源）
蓝牙鼠标 / 键盘 / 耳机使用 2.4GHz 跳频，与 WiFi 信道 1-11 频段重叠，可导致间歇性丢包。
匿名 / 隐藏 RF 源（当前快照）
无 SSID 的射频信号，可能是隐藏 WiFi、无线摄像头、微波炉、无线门铃、Zigbee 设备等。2.4GHz 频段尤其需要关注。
